# 可爱萝卜价格配置系统 🥕

这是一个基于 Python 和 PySide6 开发的桌面端自动化表格处理与价格配置工具。主要用于从总表中提取商品价格与差价信息，通过可视化界面灵活配置各店铺/商品的计价规则，并自动生成最终的报价/结算 Excel 表格。

## ✨ 核心功能

* **一键提取价格源**：自动解析 `总表.xlsx`，提取店铺、商品、价格、差价等关键数据，生成本地 JSON 配置。
* **灵活的可视化配置**：
  * 支持按“店铺”或按“商品”双向维度检索和筛选。
  * 提供 4 种计价模式的实时切换：**原价格**、**原差价**、**自定义固定价格**、**自定义固定差价**。
  * 所有的配置修改都会无感、实时保存至本地配置。
* **智能表格生成**：
  * **填充草稿**：自动解析带有合并单元格的 `草稿.xlsx`，并注入到 `模板.xlsx` 生成中间表。
  * **计算结果**：根据用户的计价模式配置，自动计算金额并生成 `最终结果.xlsx`。

```text
├── globals.py          # 全局变量定义（内存状态共享）

├── ui_form.py          # Qt UI 布局文件
├── widget.py           # 主程序入口，包含核心 GUI 逻辑与信号绑定
├── requirements.txt    # 依赖清单
├── 配置文件/             # (运行时生成) 存放提取的价格、差价和模式配置的 JSON 文件
